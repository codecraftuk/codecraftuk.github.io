codecraft
=========

Website for CodeCraft

To get started
=============

```
install ruby 1.9.3 (via RVM)
gem install bundler
git clone git@github.com:joejag/codecraft.git
cd <checked-out-repo>
bundle install
```

To run locally
=============
```
foreman start
```

To deploy to heroku
==================
```
git remote add heroku git@heroku.com:shielded-lowlands-7246.git
git push heroku master
```
