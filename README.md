codecraft
=========

Website for CodeCraft

To get started
=============

```
install ruby 1.9.3 (via RVM)
gem install bundler
clone repo
cd to workspace
run: bundle install
```

To run locally
=============
```
run: foreman start
```

To deploy to heroku
==================
```
git remote add heroku git@heroku.com:shielded-lowlands-7246.git
git push heroku master
```
